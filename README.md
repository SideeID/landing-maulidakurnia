# PT. Maulida Rafa Kurnia

PT. Maulida Rafa Kurnia adalah perusahaan yang menyediakan layanan konsultasi IT dan pengembangan perangkat lunak yang inovatif dan terpercaya.

## 🚀 Menjalankan Website

Untuk menjalankan website secara lokal:

```bash
# Install dependencies
bun install

# Jalankan development server
bun run dev
```

Buka browser di [http://localhost:3000](http://localhost:3000) untuk melihat website.

## 🌐 Website

Kunjungi website resmi kami di: [https://www.maulidakurnia.my.id/](https://www.maulidakurnia.my.id/)
